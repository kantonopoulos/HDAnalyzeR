# Show available background lists

`hd_show_backgrounds()` is used to show the available background lists
for Over Representation Analysis (ORA).

## Usage

``` r
hd_show_backgrounds()
```

## Value

A character vector of available background lists.

## Examples

``` r
hd_show_backgrounds()
#> Available background lists:
```
